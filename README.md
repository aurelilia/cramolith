# Cramolith

A pokemon-like battle MMO.

### Build

``` bash
# Run on desktop
./gradlew desktop:run

# Create jarfile
./gradlew desktop:dist
```

### Code style

Follow Kotlin official style guide: https://kotlinlang.org/docs/reference/coding-conventions.html
Max line length is 150.
